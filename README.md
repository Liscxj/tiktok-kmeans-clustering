**Belmont Summer Undergraduate Research (2025)**
This project explored algorithmic susceptibility among college TikTok users using survey data from a Kaggle dataset on student social media addiction. Using Python, I conducted exploratory data analysis, created correlation heatmaps, and applied K-means clustering (with the elbow method) to identify behavioral risk groups (high risk, at risk, and low risk).  

**Skills & Tools:**  
- Python (Pandas, Seaborn, Matplotlib, Scikit-learn)  
- Data cleaning & preprocessing  
- Correlation analysis  
- K-means clustering & model interpretation
  
**If continued:**  
I would expand the dataset, incorporate additional platforms, and explore supervised models to predict risk classification
